Xfce4 屏幕截图工具简介

本程序允许您截取整个屏幕、当前活动窗口或者一个自选区域的图像。您可以设置截图前的延时和截图之后所要执行的的动作，包括保存截图至一幅 PNG 图像文件，复制截图到剪贴板，使用另一程序打开截图，或者上传至免费的在线图像保存服务 ZimageZ。

本程序同时提供一个 Xfce 面板插件。

使用本程序

通过用户界面

要启动此程序，选择 Xfce 主菜单下的“附件”分类中的“屏幕截图”。

将显示如下窗口:

截取范围

“截取范围” 区域允许您设置要截图的屏幕范围:

  • 选择“全屏幕”选项，每次将截取整个屏幕的图像。
  • 选择“活动窗口”选项，每次将截取当前活动窗口的图像。默认是本窗口出现之前的最后一个活动窗口，或者，如果您设置了延时，将会是延时时间到达时的活动窗口。
  • “选择一个区域”选项允许您通过点击并拖拽鼠标后松开选择截取屏幕上一个矩形区域。
截取鼠标指针

“截取鼠标指针” 选项允许您选择是否截取鼠标指针。

截图前延时

“截图前延迟” 区域允许您设置按下 "OK" 按钮与截图操作进行之间的时间间隔。设置延时允许您在截图之前打开菜单、或者进行其他必要的操作以实现你想要截取的效果。

截图后操作

按下 "OK" 按钮后第二个窗口将会出现:

预览

“预览” 区域显示刚截取的图像的缩略图。

动作

“动作” 区域允许您选择要对该截图进行怎样的操作。

保存

“保存” 选项将会将截图保存为一个 PNG 图像文件。将会显示一个保存对话框，以便您选择保存的位置和文件名。

Xfce4 屏幕截图工具同样能够将截图保存到任何 GVfs 支持的远程文件系统上，例如 FTP、SAMBA、SFTP 或者通过 SSH 连接的远程计算机。您只需要通过 gvfs-connect 或者 Gigolo 连接至远程计算机，该计算机就会出现在保存对话框的左侧栏上。

复制至剪贴板

“复制至剪贴板” 选项允许您粘贴截图至另外的应用程序，例如文本处理软件。该选项仅当剪贴板管理器运行时有效。

以...打开

“以...打开” 选项保存屏幕截图至系统的缓存目录，并使用您从下拉列表中选择的应用程序将其打开。本程序将会自动检测支持图像的应用程序并将其添加至下拉列表之中。

上传至 ZimageZ

“上传至 ZimageZ” 选项将会将截图上传至 ZimageZ 这个免费的在线图像存储服务从而方便地与他人共享。ZimageZ 自动生成您上传的图像的大小两幅缩略图，可以用做指向完整大小图像的缩略图链接。

按下 "OK" 按钮后，下面对话框将会出现:

如果您未拥有一个 ZimageZ 用户名，您需要在 ZimageZ 网站上创建一个用户。该网站支持多种语言，您可以使用右上角的菜单进行切换。您必须填写本对话框中的所有项目。当您点击 "OK" 按钮后，上传就会开始。

下面的对话框将会提供您在 ZimageZ 上该幅截图的完整大小，大缩略图和小缩略图的链接，此外还提供 HTML 和 BBcode 的代码例子以便您创建指向完整大小图像的缩略图链接。

通过命令行

命令行选项允许您快速的进行屏幕截图。使用命令行操作同样允许您配置使用通常在键盘左上角的 Print Screen 按键进行屏幕截图。要这样做，请配置您桌面环境中的键盘快捷键绑定以在按下 Prt Scrn 键时以下一个或多个选项运行 xfce4-screenshooter。

命令行选项

-w 选项

-w 选项允许您截取当前活动窗口的图像。

-f 选项

-f 选项允许您截取整个屏幕的图像。

-r 选项

-r 选项允许您通过点击并拖拽鼠标后松开选择截取屏幕上一个矩形区域。

-d 选项

给定 -d 选项，后接一个正整数以设置截图前延时。此选项仅在给定 -w、-f 或者-r 选项时生效。

-s 选项

给定 -s 选项，后接一个已存在目录的路径，截图将会保存至该目录。此选项仅在给定 -w、-f 或者-r 选项时生效。

-o 选项

给定 -o 选项，后接一个程序名，截图将会保存在系统的缓存目录，然后使用该程序打开截图。此选项仅在给定 -w、-f 或者 -r 选项时生效。

-u 选项

给定 -u 选项，截图将会上传至 ZimageZ。查看上面章节以获得详细信息。此选项仅在给定 -w、-f 或者-r 选项时生效。

使用 Xfce 面板插件

添加该插件

要添加该插件至面板，请在面板上单击右键，选择 “添加新项目”。选择名为 “屏幕截图” 的插件然后添加。

配置该插件

通过右键点击该插件按钮并选择 “属性” 选项，您可以配置该截图插件。一个与主程序的对话框相似的首选项对话框将会出现。请查看上面章节对不同选项的介绍。

使用该插件

点击该面板插件按钮以截图。您可以在鼠标悬浮在插件按钮上方时使用鼠标滚轮选择不同的截取范围。
